## General info
A basic CRUD application using postgresql as the database
	
## Technologies
Project is created with:
* dotenv ^10.0.0v
* NodeJs v14.18.1
* npm v6.14.15
* Express ^4.17.1v
* Nodemom ^2.0.13v
* pg ^8.7.1v
* pg-hstore ^2.3.4v
* sequelize ^6.7.0v

	
## Setup
To run this project, install it using npm:

```
$ npm install
$ npm run dev
```
or

```
$ npm install
$ npm run start
```
